# Security Policy

## Supported Versions

We actively support and patch the latest major version of Angular Helpers.

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please **do not** open a public issue. Instead, report it privately using GitHub's built-in **Private Vulnerability Reporting** feature:

1. Navigate to the main page of the repository.
2. Under the repository name, click **Security**.
3. In the left sidebar, under "Reporting", click **Advisories**.
4. Click **Report a vulnerability** to open the draft advisory form.
5. Fill out the details of the vulnerability and click **Submit report**.

For more details, see the [GitHub documentation on private vulnerability reporting](https://docs.github.com/en/code-security/security-advisories/guidelines-for-reporting-a-vulnerability/reporting-a-vulnerability).

We will review your vulnerability report privately and strive to investigate it and provide a resolution plan within 7 days.
