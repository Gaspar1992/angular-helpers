# Support Policy

Thank you for using Angular Helpers! We want to make sure you get the help you need. Please use the following GitHub resources for support:

## 💬 GitHub Discussions (Recommended)

For questions, discussions, architectural advice, and sharing ideas, please use **GitHub Discussions**:

- **Q&A**: Ask how-to questions and get help from the community.
- **Show and Tell**: Share what you've built with Angular Helpers.
- **General**: Discuss anything related to the project.

👉 [Go to GitHub Discussions](https://github.com/Gaspar1992/angular-helpers/discussions)

## 🐛 GitHub Issues

If you have found a bug or want to request a new feature, please open an issue:

- Search existing issues first to see if it has already been reported.
- Provide a clear description, reproduction steps, and a minimal reproduction repository if possible.

👉 [Open a GitHub Issue](https://github.com/Gaspar1992/angular-helpers/issues)

## 📖 Documentation

For detailed guides, API references, and examples, please visit our documentation website:

👉 [Read the Documentation](https://gaspar1992.github.io/angular-helpers/)
